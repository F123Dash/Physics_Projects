# Physics Projects

A collection of computational physics projects spanning statistical mechanics, computational electromagnetics, fluid dynamics, and scientific machine learning.

This repository serves as a growing archive of physics simulations, numerical solvers, and data-driven modeling projects built from first principles using modern computational methods.

---

## Project Goals

- Reproduce physical phenomena through numerical simulation
- Explore advanced computational physics techniques
- Apply machine learning to scientific problems
- Develop high-performance scientific software
- Build a portfolio of research-oriented physics projects

---

## Current Projects

### 1. Ising Model

Implementation of the 2D Ising Model for studying phase transitions and critical phenomena in statistical mechanics.

#### Features

- Metropolis Monte Carlo algorithm
- Wolff Cluster algorithm
- CUDA acceleration
- Data analysis utilities
- Visualization tools

#### Topics

- Critical temperature
- Magnetization
- Susceptibility
- Phase transitions
- Finite-size effects

---

### 2. FDTD (Finite Difference Time Domain)

Electromagnetic wave propagation simulations using the Finite Difference Time Domain method.

#### Implementations

- 1D FDTD Solver
- 2D FDTD Solver
- 3D FDTD Solver

#### Topics

- Maxwell's Equations
- Wave propagation
- Electromagnetic fields
- Numerical electromagnetics
- Computational electrodynamics

---

### 3. Turbulence-CNN

A hybrid computational fluid dynamics and machine learning project focused on turbulence modeling.

#### Components

- Navier-Stokes solver
- Dataset generation pipeline
- U-Net based neural networks
- Model training and evaluation framework

#### Topics

- Computational Fluid Dynamics (CFD)
- Turbulence modeling
- Deep learning
- Scientific Machine Learning (SciML)
- Physics-informed data generation

## Installation

Clone the repository:

```bash
git clone https://github.com/F123Dash/Physics_Projects.git
cd Physics_Projects
```
Make a python venv
```python
python -m venv myenv
```

Install Python dependencies:

```bash
pip install numpy matplotlib scipy torch
```

CUDA-enabled projects require:

- NVIDIA GPU
- CUDA Toolkit
- Compatible C++ compiler
